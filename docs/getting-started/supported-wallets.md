# Supported Wallets

Whalehub supports three wallet options for connecting to the Stellar network.

| Wallet | Platform | Notes |
|--------|----------|-------|
| **Freighter** | Browser extension | Chrome, Firefox, Brave |
| **LOBSTR** | Mobile + browser | iOS, Android, web app |
| **WalletConnect** | Any compatible wallet | Scans QR code to connect |

## Connecting Your Wallet

1. Visit the Whalehub app
2. Click **Connect Wallet** in the top navigation
3. Select your wallet provider
4. Approve the connection in your wallet
5. Your AQUA and BLUB balances will appear automatically

## Important Notes

- Your wallet must be on the **Stellar public network** (mainnet)
- You need a small XLM balance for transaction fees (typically 0.01-0.1 XLM per transaction)
- You must have an **AQUA trustline** set up in your wallet before staking
- BLUB trustlines are created automatically when you first receive BLUB
